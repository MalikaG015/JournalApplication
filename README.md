# JournalApplication
I developed a Journal Application that allows users to create, edit, and manage their daily entries in a simple and organized way. The application supports core features such as adding new journal entries, updating existing ones, and deleting unwanted records. It is designed with a clean and user-friendly interface to ensure a smooth experience, while the backend efficiently handles data storage and retrieval.

To enhance security, I integrated Spring Security into the application. It provides authentication and authorization mechanisms, ensuring that only registered users can access their personal journal entries. I implemented secure login functionality, password encryption, and role-based access control to protect user data. This integration helped me gain hands-on experience with securing REST APIs and managing user sessions effectively.
